# Cat Sounds Directory

Place the following cat sound files in this directory:

- cat_start.m4a  - Recording start sound (meow)
- cat_stop.m4a   - Recording stop sound (short meow)
- cat_success.m4a - Success sound (purring)

If no files are present, system sounds will be used as fallback.

Recommended free sound sources:
- freesound.org
- pixabay.com/sound-effects
